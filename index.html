Data Collection Using OpenStreetMap

I collected data for each zip code in my dataset by using OpenStreetMap to get the nearby amenities by type, counting the distribution of amenities within a few miles of any given location. The below example just shows Irvine's data because I don't want to overload the site, but I collected data for many major cities across the country.

Takeaways

During this brief project, I learned about real estate and what factors affect rental prices. My eventual goal is to be able to use rental price projections as well as vacancy rate projections to create a more advanced Discounted Cash Flow Projection model. In order to do this, I will need to incorporate timeseries analysis into my model as well as collect better vacancy rate data.

Model Use Case

This model could be useful for predicting changes in rental prices in the future based on construction of new amenities.
